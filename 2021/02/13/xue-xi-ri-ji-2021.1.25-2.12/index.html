安装服务器版ubuntu

  • 记得自己之前安装成功了的，但是现在失败了，失败得不可思议
  • 制作u盘驱动，我使用的是固态硬盘盒，安装ubuntu18.04.5，结果显示无法获取大小什么的，很纳闷，之前直接安装成功了，现在觉得是版本问题，换了个16.04.7的，这个还可以选择语言和键盘布局，结果检测不到光盘驱动。
  • 不知道什么问题了，把上一个无法获取大小搜一下，结果说这是18.04.5的bug，我就按推荐下载18.04.1，结果可以选择语言，到选择键盘布局了就有问题了。
  • 我觉得是服务器版的系统的问题，然后我再安装win7系统，结果还是失败，真的没辙了。
  • 那就想想如何关闭桌面版的GUI界面吧，搜到一篇博客https://www.cnblogs.com/schips/p/10577464.html
  • 关闭用户界面,然后重启
sudo systemctl set-default multi-user.target

  • 本来没有抱多大的希望，因为自己之前试过的，但是这次成功了。

  • 而且达到了自己的期望，合上笔记本电脑依然可以连接，在这里找到了输入命令行好卡的原因

  • 我连接wifi远程连接好卡，接上网线上不卡，这里需要澄清一下自己之前 错怪校园网了

服务器到期

  • 阿里云还有三天到期了，准备长期使用这个服务器的，但是想想没什么必要了。
  • 因为我要试试，在自己电脑上安装nps，然后让旧电脑连接，目的是查看ip地址，路由器断电ip就会变，我在其他地方就无法登录了。

function fail()

{

        echo Error: "$@" >&2

        exit 1

}

function check_for_file()

{

        if [ ! -e "$1" ]; then

                fail "Missing file: $1"

        fi

}

z-index“不犯二”准则

  • 在看《css世界》这本书的时候，看见了这个准则，想起了自己在写网页时出现按钮由于这个属性失效的原因，即使设置很大，也可能点击不到
  • 准则内容：对于非浮层元素，避免设置z-index值。z-index值没有任何理由超过2
  • 防止出现“一山更比一山高的情况”
