一、HTML5 概述

介绍

超文本标记语言（英语：HyperText Markup Language，简称：HTML）是一种用于创建网页的标准标记语言。

后缀名

.html

HTML标签

HTML 标记标签通常被称为 HTML 标签 (HTML tag)。

HTML 标签是由尖括号包围的关键词，比如 <html>

HTML元素

一个 HTML 元素包含了开始标签与结束标签

标签 又叫“ 元素”， 例如< p> 标签 又叫 p 元素， 叫 法 不同， 意思 相同。

HTML5的优势
  • 解决跨浏览器问题
  • 部分代替了原来的JavaScript
  • 更明确了语义支持
  • 增强了Web应用程序的功能
HTML5的语法变化
  • 标签不再区分大小写
  • 元素可以省略结束标签
  • 允许省略属性值的省略
  • 允许属性值不使用引号
响应式

二、基本结构

、、三个标签用于定义总体结构以及提供简单标题信息，这些标签构成了每个网页的基本框架。

声明文档

  • 声明必须位于 HTML5 文档中的第一行,使用非常简单:
<!DOCTYPE html>

HTML5第一行必须包含它

<html>标签

指出当前文件使用的是HTML语言，必须紧跟在标识符DOCTYPE标识符后i按

<html>标签是一个容器，包含组成网页的所有标签

html 标签 的 作用 相当于 开发者 在 告诉 浏览器， 整个 网页 是 从< html > 这里 开始 的， 然后 到< /html > 结束。

<html lang="en">

可以指定语言

<head>标签

标签< head> 是一 个 容器， 包含 所有 提供 有关 网页 的 信息 的 标签， 但不 包含 将 显示 在 网页 中的 信息。

通常， 在 网页 的< head> 部分 只 使用 六个 标签， 其中 最 著名 的 是\ < title>,还有<meta>,<link>,<style>,<script>,<base>。 千万 不要 将 页面 的 文本 放在 标题（ 标签< head> 之间）。

<body>标签

<body>标签内存放文本和其他内容（链接、图片）

名称<title>

每个 HTML 页面 都 需要 一个 名称， 它 指出 了 页面 描述 的 是什么。 用户 浏览 网页 时， 这个 名称 出现 在 浏览器 的 标题 栏中。 这个 名称 被 浏览器 用来 创建 书签； 还 被 搜索 引擎 用来 建立 页面 索引。

标签< title> 放在 标签< head> 里面， 通常用于 描述 网页 的 内容

标签内的内容太长可能会被浏览器截断，可以使用多个标签存放

head 标签 内 的 meta 标签

meta 标签 又叫“ 元 信息 标签”， 是 head 标签 内 的 一个 辅助性 标签。 meta 标签 内 的 信息 不 显示 在 页面 中， 一般 用来 定义 页面 的 关键字、 页面 的 描述 等， 以 方便 搜索 引擎 蜘蛛（ 如 百度 蜘蛛、 谷 歌 蜘蛛） 来 搜索 到这 个 页面 的 信息。 通俗 点 说， meta 标签 就是 用来 告诉 搜索 蜘蛛 这个 页面 是 干嘛 的。

<meta charset="UTF-8">

\ 标签永远位于 head 元素内部。

<meta> 元素可提供有关页面的元信息（meta-information），比如针对搜索引擎和更新频度的描述和关键词。

<meta> 标签位于文档的头部，不包含任何内容。<meta> 标签的属性定义了与文档相关联的名称/值对。

  • name属性
< head>
	<!-- 网页 关键字-- >
	< meta name=" keywords" content=" 绿叶 学习 网, HTML 入门 教程, CSS 入门 教程"/>
	<!-- 网页 描述-- >
	< meta name=" description" content=" 绿叶 学习 网 是一 个 富有 活力 的 技术 学习 网 站"/>
	<!-- 本页 作者-- >
	< meta name=" author" content=" helicopter">
	<!-- 版权 声明-- > 　 　
	< meta name=" copyright" content=" 本站 所有 教程 均为 原创， 版权所有， 禁止 转载。 否则 将 追究 法律 责任。"/> 
</ head >
  • keywords：网页的关键字（可以是多个，用英文逗号隔开）
  • description：网页的描述
  • author：网页的作者
  • copyright：版权信息
  • http-equiv属性
< head>
	< meta http- equiv=" content- type" content=" text/ html; charset= gb2312"/> 
</ head>
<!-- 页面使用的字符集是gb2312，即国际汉字码 -->

< head>
	< meta http- equiv=" refresh" content=" 秒 数; url= 网址"/>
</ head>
<!-- “秒数”是一个整数，表示经过多少秒进行刷新跳转。“网址”是刷新跳转的地址 -->

学习 meta 标签 的 http- equiv 属性， 我们 只需 要 了解 它的 以下 两个 作用 就 行了。

（1） 定义 页面 所 使用 的 语言。

（2） 实现 页面 的 自动 刷新 跳 转。

自闭和标签

HTML中标签分为两种：一般标签和自闭和标签

自闭和标签不能插入任何其他标签或者文字，单个出现

  • 现在 我们 可以 总结 出来 以下 几点。

① 一般 标签 由于 有 开始 符号 和 结束 符号， 可以 在 标签 内部 插入 其他 标签 或 文字

② 自 闭合 标签 由于 没有 结束 符号， 没 办法 在内部 插入 其他 标签 或 文字， 只能 定义 自身 的 一些 属性。

  • 常见的自闭和标签

①\ < meta/> 定义 页面 的 说明 信息， 供 搜索 引擎 查看。

②< link/> 用于 连接 外部 的 CSS 文件 或 脚本 文件。

③\ < base/> 定义 页面 所有 链接 的 基础 定位。

④ < br/> 用于 换行。

⑤\ < hr/> 为 水平线。

⑥ < input/> 用于 定义 表单 元素。

⑦\ < img/> 为 图像 标签。

块元素和行内元素

HTML元素根据表现形式，可以分为两类：

  • 块元素（block)
  • 行内元素（inline）
  • 块元素

块元素默认独占一行，排斥其他元素与其位于同一行，包括块元素和行内元素。

块元素内部可以容纳其他块元素或行元素

常见块元素：

  • div
  • h1~h6
  • p
  • hr
  • ol
  • ul
  • 行内元素

行内元素默认显示状态可以与其他行内元素共存再同一行。行内元素内部可以容纳其他行内元素，但不可以容纳块元素。

常见行内元素

  • strong
  • em
  • s
  • u
  • a
  • span
  • img
  • input

有序列表

        <li>梨子</li>
        <li>香蕉</li>
    </ol>
    <h4>序号类型为小写罗马数字，起始值为3</h4>
    <ol type="i" start="3">

        <li>梨子</li>
        <li>香蕉</li>
    </ol>
</body>

五、链接

我们通过使用 <a> 标签在 HTML 中创建链接。

  • href指定跳转地址
  • title指定标题
  • target：目标窗口的打开方式

form标签属性

form标签一共有5个重要的属性，分别是name、action、method、enctype和target属性

  • name：用来给表单命名。防止提交表单时给后台造成混乱
  • action：指定表单数据提交到哪个地址进行处理
  • method：指定表单的数据使用哪一种HTTP提交方法，get或者post
  • target：指定目标窗口的打开方式。属性取值同a标签
  • enctyoe：用于指定表单信息提交的编码方式

  • text属性

  • password属性同text
  • radio属性

name和value时单选按钮必要的两个属性，必须设置

对于同一个问题不同的选项，必须设置一个相同的name属性值

  • checked属性

复选框 checkbox 不像 单选 按钮 radio， 它不 需要 设置 选项 列表 的 name， 因为 复选框 可 以多 选。

HTML 中的 复选框 是 没有 文本 的， 需要 加入 label 标签， 并且 用 label 标签 的 for 属性 指向 复选框 的 id。

  • 图片域image

既有图片效果，又有按钮效果

  • 隐藏域hidden

前台页面向后端服务器传送数据，不让用户看见

  • onclick是普通按钮的事件

  • option的属性

value:选项值

autostart属性取true或false。取值true时加载后不进行播放

hidden取值true时不显示播放器

loop取值true时无限循环播放

< embed src=" " hidden="" autostart="" loop=" "/>
